# OpenCode Installation for Notify Email

To install this skill as an OpenCode plugin:

1.  Symlink the root directory of this repository into your OpenCode plugins folder:
    ```bash
    ln -s /path/to/notify-email-skill-repo ~/.opencode/plugins/notify-email
    ```
2.  OpenCode will automatically load the skill and make it available to your agents.
3.  Ensure you have Node.js and `nodemailer` installed globally or in your agent environment.
